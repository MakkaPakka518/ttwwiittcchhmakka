## 🚀 使用方法 (Usage)

1. **下载 Forward 应用**: 确保您的设备上已安装 Forward。
2. **复制链接**: 从上方的列表中找到您需要的组件，复制其 `.js` 文件链接。
3. **导入模块**:
   - 打开 Forward App。
   - 进入 **模块添加 (Widgets)** 页面。
   - 点击右上角的 **+** 号或 **导入** 按钮。
   - 粘贴链接并确认即可。
